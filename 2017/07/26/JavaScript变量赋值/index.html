JavaScript中数据类型：

ECMAScript变量包含两种不同类型的值：基本数据类型和引用类型。

- 基本数据类型

保存在栈内存中的简单数据段
Number;String;Boolean;Undefined;Null

- 引用数据类型

保存在堆内存中的对象，意思是，变量中保存的实际上只是一个指针，这个指针指向内存堆中实际的值；
Function;Array;Date;RegExp;Error;Map;Set……
